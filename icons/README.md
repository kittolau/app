# PWA Icons Instructions

## Quick Setup

1. Open `icon-generator.html` in your browser
2. Click the "Download" button under each icon
3. Save each icon to this `/icons/` folder with the exact name shown (e.g., `icon-192x192.png`)

## Required Icons

The following icons are required for full PWA support on iOS and Android:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png (Android minimum)
- icon-384x384.png
- icon-512x512.png (Android recommended)

## Optional Screenshots

For enhanced app store listings, you can also add:
- screenshot-wide.png (1280x720) - for desktop/tablet
- screenshot-mobile.png (750x1334) - for mobile devices

## Alternative: Use Your Own Icons

If you prefer custom icons:
1. Create PNG images in the sizes listed above
2. Ensure they have transparent or white backgrounds
3. Make them recognizable at small sizes
4. Use consistent branding across all sizes
